## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/ChamathDilshanC/PortFolio-NextJs.git
cd PortFolio-NextJs
````

### 2️⃣ Install Dependencies

```bash
npm install
# or
yarn install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

> Visit **[http://localhost:3000](http://localhost:3000)** to view your site locally.

### 4️⃣ Build for Production

```bash
npm run build
npm start
```
